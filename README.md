## The project aims to improve knowledge


### Installation

```
web
$ npm install -g create-react-app
$ npm create-react-app  “cadastro-login”
$ npm start

```

```
server
$ npm init -y
$ npm install express
$ npm instal nodemon
$ npm start

```

```
mobile
$ npm install --global expo-cli
$ expo init mobile
$ npm start

```
